HOW TO USE THE AUDIOMANAGER TO MAKE SOUND DESIGN:
 * add your sound to the sound[] of the prefab AudioManager
 * type : "FindObjectOfType<AudioManager>().Play("name of your sound")" in the script, when you want the sound to be played
 * if you want to add a background music that is present throughout the whoel game, put it in the start method of the audiomanager.cs file and loop it
